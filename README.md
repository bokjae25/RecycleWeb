# 🕹️ 재활 치료 웹 게임 (Rehab Web Game)

웹 브라우저에서 즐기는 재활 치료 보조 게임 프로젝트입니다.
환자가 재미있게 반복 운동을 수행하도록 돕고, 진행 상황을 기록/시각화하는 것이 목표입니다.

## 🎯 프로젝트 목표

- 손/팔 등 상지 재활 운동을 게임 형식으로 유도
- 별도 설치 없이 브라우저(PC/태블릿)에서 바로 실행
- 운동 결과(점수, 반응 속도, 정확도)를 기록하여 진행 상황 확인
- 노인/환자 친화적 UI (큰 글씨, 큰 버튼, 높은 대비)

## 🎮 게임 아이디어 (1차 후보)

| 게임 | 재활 목표 | 조작 방식 |
|------|-----------|-----------|
| 풍선 터뜨리기 | 손-눈 협응, 반응 속도 | 마우스 클릭 / 터치 |
| 궤적 따라 그리기 | 손목·팔 운동 범위, 정밀 제어 | 드래그 |
| 순서 기억 게임 | 인지 재활 (기억력) | 클릭 / 터치 |
| 분리수거 게임 | 인지 + 조준 (드래그 앤 드롭) | 드래그 |

각 게임은 난이도 조절(속도, 크기, 개수)이 가능하도록 설계합니다.

## 🛠️ 기술 스택

- **프론트엔드**: HTML / CSS / JavaScript (Vanilla → 필요 시 확장)
- **그래픽**: Canvas API
- **데이터 저장**: localStorage (1차) → 추후 서버/DB 확장 가능
- **배포**: GitHub Pages

## 🗓️ 개발 로드맵

### 1단계: 기반 구축
- [ ] 프로젝트 구조 및 메인 화면(게임 선택 메뉴) 제작
- [ ] 공통 UI 컴포넌트 (버튼, 점수판, 타이머)

### 2단계: 첫 게임 구현
- [ ] 풍선 터뜨리기 게임 (클릭/터치 기반)
- [ ] 난이도 설정 (쉬움/보통/어려움)
- [ ] 점수 및 결과 화면

### 3단계: 게임 확장
- [ ] 궤적 따라 그리기 게임
- [ ] 순서 기억 게임
- [ ] 분리수거 드래그 게임

### 4단계: 기록 및 통계
- [ ] localStorage에 플레이 기록 저장
- [ ] 날짜별 점수/정확도 그래프

### 5단계: 마무리
- [ ] 접근성 개선 (글씨 크기, 색 대비, 소리 피드백)
- [ ] GitHub Pages 배포
