# 🕹️ 재활 치료 웹 게임 (Rehab Web Game)

웹 브라우저에서 즐기는 재활 치료 보조 게임 프로젝트입니다.
환자가 재미있게 반복 운동을 수행하도록 돕고, 진행 상황을 기록/시각화하는 것이 목표입니다.

## 🎯 프로젝트 목표

- 손/팔 등 상지 재활 운동을 게임 형식으로 유도
- 별도 설치 없이 브라우저(PC/태블릿)에서 바로 실행
- 운동 결과(점수, 반응 속도, 정확도)를 기록하여 진행 상황 확인
- 노인/환자 친화적 UI (큰 글씨, 큰 버튼, 높은 대비)

## 🎮 게임 아이디어 (1차 후보)

| 게임 | 재활 목표 | 조작 방식 |
|------|-----------|-----------|
| 풍선 터뜨리기 | 손-눈 협응, 반응 속도 | 마우스 클릭 / 터치 |
| 궤적 따라 그리기 | 손목·팔 운동 범위, 정밀 제어 | 드래그 |
| 순서 기억 게임 | 인지 재활 (기억력) | 클릭 / 터치 |
| 분리수거 게임 | 인지 + 조준 (드래그 앤 드롭) | 드래그 |

각 게임은 난이도 조절(속도, 크기, 개수)이 가능하도록 설계합니다.

## 🛠️ 기술 스택

- **프론트엔드**: HTML / CSS / JavaScript (Vanilla → 필요 시 확장)
- **그래픽**: Canvas API
- **데이터 저장**:
  - 1차: localStorage (오프라인 임시 저장)
  - 2차: **Supabase** (PostgreSQL + 인증 + REST API) — 재활 기록을 서버에 저장
- **인증/권한**: Supabase Auth (환자 / 의료진 역할 구분)
- **배포**: GitHub Pages (프론트엔드) + Supabase (데이터)

## 👥 사용자 역할

| 역할 | 할 수 있는 것 |
|------|----------------|
| 환자 | 게임 플레이, 자신의 기록만 조회 |
| 의료진 | 담당 환자 목록 조회, 각 환자의 진행 현황·그래프 확인 |

## 🩺 데이터 흐름 & 의료진 대시보드

```
[환자] 게임 플레이 → 결과 기록
   │  (오프라인 시 localStorage 임시 저장)
   ▼
[Supabase DB] sessions 테이블에 업로드
   │
   ▼
[의료진 대시보드] 로그인 → 담당 환자 선택 → 진행 그래프/통계 확인
```

### 예상 데이터 모델 (Supabase)

- **profiles**: id, 이름, 역할(patient/clinician), 담당 의료진 id
- **sessions**: id, 환자 id, 게임 종류, 난이도, 점수, 정확도, 반응속도(ms), 플레이 시간, 생성일시
- **RLS(행 수준 보안)**: 환자는 본인 기록만, 의료진은 담당 환자 기록만 접근

> ⚠️ **개인정보/의료정보 주의**: 실제 환자 데이터를 다루므로 최소 정보만 수집하고,
> Supabase RLS로 접근을 엄격히 제한합니다. 실제 서비스 전 관련 법규(개인정보보호법 등) 검토 필요.

## 🗓️ 개발 로드맵

### 1단계: 기반 구축
- [ ] 프로젝트 구조 및 메인 화면(게임 선택 메뉴) 제작
- [ ] 공통 UI 컴포넌트 (버튼, 점수판, 타이머)

### 2단계: 첫 게임 구현
- [ ] 풍선 터뜨리기 게임 (클릭/터치 기반)
- [ ] 난이도 설정 (쉬움/보통/어려움)
- [ ] 점수 및 결과 화면

### 3단계: 게임 확장
- [ ] 궤적 따라 그리기 게임
- [ ] 순서 기억 게임
- [ ] 분리수거 드래그 게임

### 4단계: 로컬 기록 및 통계
- [ ] localStorage에 플레이 기록 저장
- [ ] 날짜별 점수/정확도 그래프

### 5단계: 서버 연동 (Supabase)
- [ ] Supabase 프로젝트 생성 및 profiles/sessions 테이블 설계
- [ ] 환자/의료진 로그인(회원가입) 구현
- [ ] 게임 결과를 sessions 테이블에 업로드 (오프라인 시 localStorage → 온라인 복귀 시 동기화)
- [ ] RLS(행 수준 보안) 정책 설정

### 6단계: 의료진 대시보드
- [ ] 의료진 로그인 후 담당 환자 목록 화면
- [ ] 환자별 진행 현황 그래프 (기간별 점수·정확도·반응속도)
- [ ] 최근 활동/미접속 알림

### 7단계: 마무리
- [ ] 접근성 개선 (글씨 크기, 색 대비, 소리 피드백)
- [ ] 개인정보 보호 점검
- [ ] GitHub Pages 배포
